group by 和 having   

select a,count(*)  
from b   
group by a;    

将数据按照a分组，分别统计每类的个数      
group by 必须出现在where之后 order by 之前   

where过滤行，而having过滤分组  
where在数据分组前进行过滤，having在数据分组后进行过滤，where排除的行不包括在分组中，会影响having子句中的分组  

select a,count(*)  
from b   
group by a  
having count(*)>2;  

使用group by时也要使用order by来保证数据的顺序   

select    
from    
where    
group by   
having    
order by   
limit
